# AI Email Helper App

Select a model (GPT-4o mini or GPT-4.1) to shorten, lengthen, or change the tone of your email.

## Description

* Edit emails tab: Use AI to refine a single email.
* Generate tab: Run your selected model on an entire dataset.
* View Analysis tab: Compare the evaluation results of the two models on one user action (shorten, lengthen, change tone).

## Getting Started

### Installation

1. Get free API keys for GPT-4o mini and GPT-4.1, or use your own.
2. Clone the repository
```
git clone https://github.com/sruthiperu/email_helper_app.git
```
3. Save your API keys in a .env file. Ensure this file is added to the .gitignore file.
4. Install dependencies
```
pip install requirements.txt
```

## Execution

### Run the app

```
streamlit run app.py
```
